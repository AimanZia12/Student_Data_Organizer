
#Student Data Organizer
## This is a Student Data Organizer program based on python which is built to manage the collection of students' Data.
### In this program I have built an organizer to store and manage the student information like student name, student age, student id, student birth year, student grade and student subjects.
 
### Features-

1. **Collection and Conversion:**  Converted the student age into integer, and student id and birth year into tuple and collected the subjects in set and collected all the student information using key-value pairs through dictionary.
2. **del Keyword:**  Applied del keyword to delete the information of a student.
3. ** Update:** Updated the student information by converting the tuple values into list.

### Workflow-

1. As soon as, the program starts the user gets 6 options-
a. Add a Student Detail.
b. Display all Student Details.
c. Update all Students Details.
d. Delete all Student Details.
e. Display all Subjects Offered to Students,
f. Exit the program.

2.Taken the input from the user about the number of student whose values have to be inserted like all the important information of student like name, age, id, grade, birth year, subjects. 
3. Stored different informations in different types like in lists, tuples, sets and dictionary.
4. Displayed the whole information of students.
5. Update the student information by giving specific student id and and updated the information like new name, new age, new dob, new grade.
6. Deleted the student information by giving specific student id and deleted his/her information.
7. Displayed all Subjects Offered to the Students.
8. Exit the program, after completing all tasks.

### Technology Used-

Python 3 

### Concepts Used-

1. While Loops
2. if-elif-else 
3. Nested loops
4. Type Casting.
5. Changing the mutability
6. del keyword
7. Joining of the elements of set
8. break, pass and continue statements
